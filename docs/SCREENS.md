# Rozy screen lists (MVP)

## Passenger app — Rozy

| Screen | Purpose |
|--------|---------|
| Splash | Brand load |
| Onboarding (optional) | 2–3 slides |
| Phone login | OTP entry |
| Home / map | Pickup pin, destination, ride type selector |
| Place search | Autocomplete + recent + POI |
| Fare estimate | Before confirm |
| Finding driver | Searching animation |
| Driver assigned | Photo, plate, ETA, call masked |
| Trip PIN display | Show PIN to driver |
| Active trip | Live map, SOS, share trip |
| Trip complete | Fare summary, rate trip, report issue |
| Trip history | List |
| Profile | Name, phone |
| Notifications | In-app list |

**Theme:** cream background, gold CTAs, white cards

## Driver app — Rozy Driver

| Screen | Purpose |
|--------|---------|
| Splash | Brand load |
| Phone login | OTP |
| Choose mode (once) | Boda vs Car (Basic/XL) — **immutable after submit** |
| Verification wizard | NIN, ID photos, selfie, permit, vehicle, insurance |
| Pending approval | Waiting state |
| Home / map | Go online toggle, wallet badge |
| Wallet | Balance, recharge (MTN/Airtel), history |
| Recharge | Amount + provider |
| Incoming request | Accept / reject countdown |
| Navigate to pickup | Map + passenger info |
| At pickup | Enter passenger PIN |
| Active trip | Navigation, complete trip |
| Trip complete | Fare earned, fee deducted |
| Documents | View status, re-submit if expired |
| Profile | Verified vehicle snapshot |
| Notifications | |

**Theme:** dark gold header, cream body, black premium wallet cards
